# 04 - Параметры устройства и конфигурация

## Основные параметры AdEM

Приложение читает и записывает множество параметров устройства. Каждый параметр имеет числовой идентификатор (Item Number).

### Конфигурационные параметры

| Параметр | Описание |
|----------|----------|
| serialNumber | Серийный номер устройства |
| serialNumberPart2 | Вторая часть серийного номера |
| firmwareVersion | Версия прошивки |
| firmwareChecksum | Контрольная сумма прошивки |
| productType | Тип продукта |
| date | Текущая дата |
| time | Текущее время |
| gasDayStartTime | Время начала газового дня |
| dateFormat | Формат даты |
| batteryType | Тип батареи |
| sealStatus | Статус пломбы |

### Параметры измерений

| Параметр | Описание |
|----------|----------|
| uncVolUnit | Единицы некорректированного объема |
| corVolUnit | Единицы корректированного объема |
| uncVolDigits | Разрядность некорр. объема |
| corVolDigits | Разрядность корр. объема |
| pressUnit | Единицы давления |
| tempUnit | Единицы температуры |
| meterSize | Размер счетчика |

### Параметры интервального лога

| Параметр | Описание |
|----------|----------|
| intervalLogType | Тип интервального лога |
| intervalLogInterval | Интервал записи |
| intervalField5-10 | Дополнительные поля |

## Недоступные параметры по типам устройств

Каждый тип AdEM имеет свой набор недоступных параметров:

### AdEM S
Не поддерживает: 8, 9, 10, 11, 13, 14, 26, 27, 28, 53, 54, 55, и др.

### AdEM T
Не поддерживает: 8, 9, 10, 11, 13, 14, 54, 55, 63, 87, 98, и др.

### AdEM TQ
Не поддерживает: 8, 9, 10, 11, 13, 54, 55, 63, 87, 88, и др.

### AdEM PTZ
Расширенная поддержка параметров давления и температуры.

## Защищенные пломбой параметры

Следующие параметры защищены пломбой и не могут быть изменены:
- 0, 2, 13, 34, 87, 90, 92, 109-112, 122, 123, 138, 147, и др.

## Кэширование

Приложение использует кэширование для оптимизации:

### ConfigCache
- gasDayStartTime
- dateFmt, timeFmt
- lastSaveDate, lastSaveTime
- backupIdxCounter
- batteryType
- isSealed

### MeasureCache
- meterSize
- uncVolUnit, corVolUnit
- pressUnit, tempUnit
- intervalType, intervalSetting
- и другие единицы измерения

## Конвертация единиц

| Конвертация | Коэффициент |
|-------------|-------------|
| kPa -> PSI | 0.145038 |
| kPa -> Bar | 0.01 |
| kPa -> inH2O | 4.018598072 |
| C -> F | (C * 1.8) + 32 |
